# **Connecting Rod Assembly in SolidWorks:**



### **Description:**



This project presents a detailed 3D CAD model of a Connecting Rod Assembly developed using SolidWorks 2025. The assembly consists of two main components: the connecting rod and the connecting cap. It is designed to represent a real-world mechanical component used in internal combustion engines, responsible for transmitting motion between the piston and crankshaft. The model is created with proper dimensions, realistic geometry, and engineering considerations. All parts are assembled using appropriate mates to ensure correct alignment and functionality.



### **Features:**



* Material Specified: AISI 4340 Steel, annealed.
* 3D Features: Box-Extrude, Cut-Extrude, Fillets.
* Mates Used: Concentric Mate, Coincident Mate.
* Fully Constrained Assembly.
* Accurate Dimensioning.
* Realistic Mechanical Design.
* Clean Feature Tree Structure.
* Rendered Visualization Images.
* Ready For FEA Analysis.



### **Components Files:**



* Connecting\_rod.SLDPRT
* Connecting\_rod\_cap.SLDPRT
* Connecting\_assembly.SLDASM
