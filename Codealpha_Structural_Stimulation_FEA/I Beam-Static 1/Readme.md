# Structural Simulation (FEA) of I-Beam:



### Description:

This project focuses on performing a Finite Element Analysis (FEA) on an I-beam to evaluate its structural performance under different loading and boundary conditions.



The objective is to analyze how the beam behaves when subjected to external forces by studying stress distribution, deformation, and factor of safety. The simulation helps in understanding the strength and reliability of the component in practical engineering applications.



### Features:

* Finite Element Modeling of I-Beam
* Application of Different Loads
* Implementation of Boundary Conditions
* Stress Distribution Analysis (Von Mises Stress)
* Deformation Analysis
* Factor of Safety Evaluation
* Simulation Result Visualization



### Simulation Details:

* Analysis Type: Static Structural Analysis.
* Component: I-Beam (3D Solid Model)
* Material: AISI 304 Stainless Steel.
* Loads Applied: A force of 1,000 kgf applied to the top flange via split line distribution.
* Boundary Conditions: Fixed supports applied to the bottom faces of the beam ends.
* Mesh Type: Solid Mesh (High Quality).
* Outputs:
* Von Mises Stress Resultant.
* Displacement (URES).
* Equivalent Strain (ESTRN).
* Factor of Safety (FOS).



### Results \& Observations:

* Maximum stress occurs at critical regions such as supports and load application points.
* Deformation increases with load magnitude and depends on boundary conditions.
* Factor of safety indicates whether the design is safe under given loading conditions.
* The simulation validates the structural efficiency of the I-beam.



### Tool Used:

SolidWorks 2025 (Simulation).
